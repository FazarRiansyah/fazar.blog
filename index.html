Bagaimana #JagaDokumen Melindungi Privasi Anda?

Semua pemrosesan dilakukan sepenuhnya di browser menggunakan Web APIs dan JavaScript. File Anda tidak pernah meninggalkan perangkat. Tidak ada backend, tidak ada tracking, tidak ada penyimpanan eksternal.

🔐

Cara Kerja #JagaDokumen

Transparansi penuh tentang keamanan Anda

🌐

File API, Canvas API & pdf-lib

Browser modern memiliki API bawaan untuk membaca, memproses, dan memanipulasi file. Library pdf-lib berjalan sepenuhnya di browser. Semua terjadi di memori RAM komputer Anda.

🔒

Tidak Ada Transmisi Data

Setiap operasi bersifat lokal. Tidak ada request ke server luar. Anda bisa memverifikasi ini dengan membuka DevTools → Network.

📵

Bisa Offline

Setelah halaman termuat, Anda dapat memutus koneksi internet dan semua fitur tetap berfungsi normal.

🗑️

Data Hilang Saat Halaman Ditutup

Tidak ada data yang disimpan secara permanen. Refresh halaman = semua data hilang dari memori.
